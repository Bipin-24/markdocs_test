# Zeenea Administration

This section provides information to help administrators **configure, manage, and maintain** Zeenea environments.


{% callout type="check" %}
- **[Manage Users](/docs/Zeenea_Administration/zeenea-managing-users):**  The Manage Users role is often shared among multiple collaborators to cover the technical and functional responsibilities of the platform.

- **[Manage Groups](/docs/Zeenea_Administration/zeenea-managing-groups):**  Groups replace Permission sets to manage user permissions in Zeenea. Existing permission sets have already been automatically migrated to Groups with the same descriptions and scopes.

{% /callout %}
