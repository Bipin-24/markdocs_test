# {% $markdoc.frontmatter.title %} 

Zeenea **Studio** and **Explorer** applications can run on any computer with an internet connection and support the following web browsers.
{% table %}
* Browser
* Notes
---
* Google Chrome™ (most recent stable version)
* Zeenea regularly tests and supports the latest stable version of Chrome. No specific configuration is required.
---
* Mozilla® Firefox® (most recent stable version)
* Zeenea regularly tests and supports the latest stable version of Firefox. No specific configuration is required.
---
* Microsoft® Edge®
* Use the latest version supported by Zeenea and ensure all Microsoft software updates are applied.
---
* Apple® Safari® (versions 11.x and 12.x on macOS)
* No specific configuration is required for Safari.
{% /table %}

## Browser Recommendations


{% callout type="info" %}
The following recommendations apply to all supported browsers.
{% /callout %}

- JavaScript, cookies, and **TLS 1.0 or later** must be enabled.
- For the best user experience, a minimum screen resolution of **1440 × 900** is recommended.
